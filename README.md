# LLM Hallucination Reduction Project

This project evaluates techniques to reduce hallucinations in large language models using the TruthfulQA benchmark.

## Techniques
- Baseline prompting
- Prompt constraints
- Self-consistency
- Self-correction
- Retrieval-Augmented Generation (RAG)

## Setup
1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Run Ollama locally
4. Configure `.env`
